# Get Started with David UI

David UI is a free and open-source components library for ReactJS and Tailwind CSS inspired by Material Design.

---

## Installation

Learn how to use `@material-tailwind/react` components from this documentation to quickly and easily create elegant and flexible pages using Tailwind CSS.

`@material-tailwind/react` is working with Tailwind CSS classes and you need to have Tailwind CSS installed on your project - <Link href="https://tailwindcss.com/docs/installation/framework-guides?ref=material-tailwind" target="_blank">Tailwind CSS Installation Guide</Link>.

<br />

<CodePreview>
```bash
npm i @material-tailwind/react@beta
```
</CodePreview>

---

## TailwindCSS Configurations

Once you installed `@material-tailwind/react` you need to add `mtConfig` to your `tailwind.config.js` file.

<br />

<CodePreview>
```js
// highlight(2,7,9)
import type { Config } from "tailwindcss";
import { mtConfig } from "@material-tailwind/react";

const config: Config = {
  content: [
    ...
    "./node_modules/@material-tailwind/react/**/*.{js,ts,jsx,tsx}"
  ],
  plugins: [mtConfig],
};

export default config;
```
</CodePreview>

<br />

For more guides on how to use `@material-tailwind/react` with different frameworks, check the <Link href="#frameworks-integration">Frameworks Integration</Link> section.
